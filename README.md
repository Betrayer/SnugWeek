# SnugWeek

A cozy weekly planner-journal: notebook-style week spread, drag-and-drop planning, day trackers, habit grid, week notes. Offline-first with automatic sync.

## Development

```
npm install
npm run dev
```

Copy `.env.local.example` to `.env.local` and fill in the Firebase web app config.

## Scripts

- `npm run dev` — dev server
- `npm run build` — type-check + production build
- `npm run lint` — ESLint
- `npm run format` — Prettier over `src`
